Hexo文章置顶

NexT主题本身支持置顶,只需要在文章的头部添加sticky: value 字段就可以支持排序,NexT根据sticky的值来进行排序,但是显然这样并不够通用,换一个主题就无法排序了,通用的做法类似NexT,在文章头部添加字段,按照字段的值进行排序,通过这样的方式来进行置顶.
